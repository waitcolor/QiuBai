糗百简易版
======

##项目简介
利用 糗事百科接口及其资源实现其主要功能


##主要功能

###List主界面

* 加载专享糗百数据
* 点击进入糗百详情查看评论
* 支持下拉刷新以及上拉加载更多
* 支持离线缓存
* 清除本地图片缓存
